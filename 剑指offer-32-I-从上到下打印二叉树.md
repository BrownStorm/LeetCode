# 从上到下打印二叉树

从上到下打印出二叉树的每个节点，同一层的节点按照从左到右的顺序打印。

 

例如:
给定二叉树: [3,9,20,null,null,15,7],

    	3
       / \
      9  20
        /  \
       15   7

返回：

```
[3,9,20,15,7]
```



## 思考

因为是从上至下从左至右按层打印，所以是按照：“根节点、【左孩子、右孩子】、【左孩子的左孩子、左孩子的右孩子】、【右孩子的左孩子、右孩子的右孩子】”这个顺序打印。很容易看出是先进先出，使用了**队列**的数据结构。

## 复杂度

- 时间复杂度：$O(N)$。每个节点便利一遍
- 空间复杂度：$O(N)$。最差情况下，即为平衡二叉树是，最多有$N/2$个节点同时在队列中
